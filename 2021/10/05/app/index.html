macOS上好用的软件

知识管理

  • Obsidian，一款开源笔记工具，提供反向链接和自动知识图谱生成，市面上有很多类似的软件，个人感觉这款功能齐全、UI不错，并且它免费！
    备份问题：因为Obsidian以markdown的形式本地化存储笔记，需要自己备份：可以安装插件Obsidian Git，定时同步到Github仓库。
  • Zotero，文献管理软件，以下是我常用到的场景:

    • 下载论文
      （安装safari-zotero插件后）arxiv等网页上可以自动抓取

    • 论文存储和分类

      • 自定义分类整理
      • 高级检索
      • 标记阅读状态
      • 排序按日期、显示会议

    • 个人论文写作

      • 插入参考文献
      • 调整参考文献格式

    • 安装zotero插件
      把 .xpi 文件下载到本地，然后在 Zotero中点选 “Tools → Add-Ons”, 把 .xpi 文件拖进窗口。

编程

  • Gist：代码片段收集和管理，可以与其他人共享代码片段
    • 每个Gist都是一个Git库，有版本历史，可以被fork/clone
    • Gist分为公开的和私有的，私有的Gist不会在Gist主页显示，也无法用搜索引擎搜索到，但它的链接是开放访问的
  • VScode ：有丰富的插件生态，能够满足大部分开发需求。

效率工具

  • Alfred（macOS系统）：可以定制你想要的任意工作流程，使一些需要很多步骤的复杂工作变成简单的自动化流程，大大提高你的学习工作效率；双击command键就可以打开Alfred搜索框进行输入。
    常用的功能：
    • 快速打开某个App：输入app的前面几个字母，就能直接打开
    • web Search：直接在搜索框输入想搜索的内容，选择搜索引擎（或默认谷歌）就可以打开浏览器并出现搜索结果；
    • 自定义网站搜索（用的最多）：可以快速打开自己常用的网站并在站内进行搜索；以查找论文和代码的PapersWithCode网站为例:

  • iTerm2：一款完全免费的，专为 Mac OS 用户打造的命令行应用。可以默认开机启动，开启全局唤醒快捷键
    实用功能推荐：
    • 主题换肤、透明背景，很多方案选择；
    • 快速打开远程终端：iTerm2可以为不同的终端设置快捷键。例如我给我的云服务器专门设置了一个终端，可以直接用F4打开云服务器的命令行界面。
  • shadowRocket，小火箭，翻墙用，一个apple账号买了之后可以同时在手机和电脑上下载。
